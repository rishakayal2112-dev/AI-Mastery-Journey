Day 3 Progression (basic to intermediate topics)
=== = =========== ====== == ============ =======

1. List Methods
   ~ append() - to add a new data value in the end of the list.
   ~ insert() - to add a data value in the list, in a specified position using index.
   ~ remove() - to remove a data from the list.
   ~ pop() - to remove the last data in the list. Whether it was newly added or older.
   ~ sort() - to arrange the data in ascending order.
   ~ reverse() - to arrange the data in descending order.

2. Dictionary Methods
   ~ keys() - used to display the key components like; name, age. without the direct answers like our name.
   ~ values() - used to display the values like: kayal,22 without the keys.
   ~ items() - used to display the overall items stored in a dictionary. like: Name:kayal, age:22.
   ~ get() - to get the specified value using its keys.
   ~ update() - add a new value with a key in the existing dictionary. 

3. Modules
   ~ Math Module - to perform several math problems by importing it(math module)eg:import math.
      #sqrt - to find the sqrt value.
      #pi - to find pi value.
      #pow - to find power of a value.
   ~ Random Module - to print a random number between specified numbers.

4. Exception Handling - protecting the code from crashing using try-except.

5. Object-Oriented Programming (OOP) - To create a class and onject in it.
